## 📖 簡介

> **本檔案內容由 AI 輔助生成，作者不對內容的絕對準確性負責。**

本專案提供一組適用於 Minecraft Bedrock Edition（基岩版 / MCPE）的高品質皮膚包，並附帶相關的打包與加密工具。
本專案採用**「快取替換技術（Skin Pack Injection）」**，透過修改與加密 `skins.json`、`.lang` 及皮膚圖檔，將原本枯燥的官方市集皮膚包「移花接木」替換成我們自訂的精美皮膚，同時能在多人連線與伺服器中正常顯示。

## 📁 檔案結構說明

- `pack/` — 核心資源資料夾，包含未加密的自訂檔案：`skins.json`, `geometry.json`, 皮膚圖檔與 `texts/`（語言檔）。
- `工具檔` — 用於將明文檔案就地轉換為遊戲可讀取加密格式、並生成簽章的程式。
- `.gitignore` — 標記了開發過程中的暫存檔（如 `temp/` 等），這些檔案不會包含在正式發佈中。

## ⚙️ 安裝與替換步驟

請嚴格按照以下步驟操作，以確保皮膚包能成功載入遊戲。

### 步驟 1：定位官方快取資料夾
在 Windows 11 中，請開啟檔案總管，前往以下快取路徑尋找你想要「替換/覆蓋」的官方皮膚包（你需要先在遊戲市集中下載過該皮膚包）：
> **預設路徑：** 
> `C:\Users\你的使用者名稱\AppData\Roaming\Minecraft Bedrock\premium_cache\skin_packs\`
> *(資料夾名稱通常為一串亂碼，例如 `u9bQ7FAljNM=`)*

### 步驟 2：選擇與清理目標資料夾（⚠️ 非常重要）
1. **挑選目標**：建議選擇一個你在遊戲內容易辨識的官方皮膚包（例如 **"1st Birthday Skin Pack"**），這樣替換成功後，在遊戲的更衣室裡會比較好找。
2. **保留身份證**：進入該官方皮膚包的亂碼資料夾後，**務必保留官方原本的 `manifest.json`**（這是官方包的身份證，絕對不可以刪除）。
3. **清空內容**：將資料夾內除了 `manifest.json` 以外的**所有官方檔案全部刪除**。

### 步驟 3：植入自訂皮膚檔案
將本專案 `pack/` 資料夾內的所有內容（你的自訂皮膚、`.lang`、`skins.json` 等），全部複製並貼上到剛剛清理好的官方皮膚包亂碼資料夾內。
*(此時該目標資料夾內，應該只有官方的 `manifest.json` 以及你剛貼上的自訂檔案)*

### 步驟 4：使用加密工具就地處理
由於基岩版會對檔案進行加密校驗，若直接啟動遊戲將無法讀取。請使用本專案附帶的加密工具來完成最後的偽裝：
1. 執行專案內的加密工具 `encryptor.exe`。
2. 依照工具的提示，將**「【步驟 2】的官方皮膚包亂碼資料夾路徑」**貼給加密工具。
3. 工具會自動處理好該資料夾內所有檔案的加密程序，並就地生成所需的數位簽章（如 `signatures.json`）。

### 步驟 5：重啟遊戲
啟動 Minecraft，進入更衣室（Dressing Room）。原本的官方皮膚包現在應該已經顯示為本專案的自訂皮膚與自訂名稱（由 `.lang` 檔案控制）了！

## 🛠️ 如何貢獻與新增皮膚

歡迎熟悉修改機制的玩家提交 PR (Pull Request)：
1. 請在 `pack/skins.json` 中新增你的皮膚設定陣列，並將對應的圖檔放入 `pack/` 的正確路徑。
2. 修改 `pack/texts/` 中的對應語言檔（`.lang`），設定好你的皮膚顯示名稱。
3. 測試加密並放入遊戲確認無誤後，再發起 PR。
*(請勿將 `.gitignore` 排除的暫存檔或無關的加密產出物提交到 PR 中)*
