Overview

This repository contains a skin pack for Minecraft Bedrock / Pocket Edition (MCPE). It is intended primarily for players; users who can edit files and upload images are welcome to contribute.

Key points

- Use only the files inside the `pack/` folder as the authoritative game assets. Do not rely on ignored or temporary files elsewhere in the repository.
- Check the project's `.gitignore` (examples: `fix_names.ps1`, `list_files.bat`, `temp/`, `pack -*/`)—those files are intentionally excluded from version control and should not be used as distributable assets.
- Never overwrite the game's original (official) `manifest.json` with the one from this repository. Always back up the target folder's `manifest.json` and exclude it when copying files.

Installation (summary)

1. Back up the target resource pack folder (especially `manifest.json`).
2. Copy the contents of `pack/` into the target resource pack folder, but exclude or retain the target's original `manifest.json` (do not replace it with this repo's manifest).
3. Launch the game and enable the resource pack in Settings → Global Resources.

Windows example (PowerShell):

```powershell
$src = 'C:\path\to\MCPE-Skin-Pack\pack\'
$dst = 'C:\path\to\minecraft\resource_packs\YourTargetPack\'
robocopy $src $dst /E /XF manifest.json
```

Contributing new skins

- Edit `pack/skins.json` and the language files in `pack/texts/` to add new entries and display names, and add your texture files to an appropriate path inside `pack/`.
- Follow the structure of the existing `skins.json` entries as examples.
