개요

> **이 파일은 AI에 의해 생성되었습니다. 작성자는 내용에 대해 책임을 지지 않습니다.**

이 저장소는 Minecraft Bedrock / Pocket Edition 용 스킨 팩을 제공합니다. 주로 플레이어를 대상으로 하며, 파일을 편집하고 이미지를 업로드할 수 있는 기여자도 환영합니다.

중요 안내

- 게임에 사용되는 정식 자산은 `pack/` 폴더만을 기준으로 사용하세요. 리포지터리의 무시된 임시 파일이나 도구는 사용하지 마세요.
- 루트의 `.gitignore` 를 확인하세요(예: `fix_names.ps1`, `list_files.bat`, `temp/`, `pack -*/`). 이러한 항목은 버전 관리에서 제외되어 배포용으로 사용하면 안 됩니다.
- 게임의 공식 `manifest.json` 을 이 저장소의 `manifest.json` 으로 덮어쓰지 마세요. 대상 폴더의 `manifest.json` 을 백업하고 복사 시 제외하세요.

설치 (요약)

1. 대상 리소스 팩 폴더를 백업하세요(`manifest.json` 특히 주의).
2. `pack/` 의 내용을 대상 폴더로 복사하되, 대상의 `manifest.json` 은 덮어쓰지 마세요.
3. 게임을 실행하고 설정 → 리소스 팩에서 활성화하세요.

기여 방법

- 새 스킨을 추가하려면 `pack/skins.json` 과 `pack/texts/` 언어 파일을 수정하고, 텍스처 파일을 `pack/` 의 적절한 위치에 추가하세요.
